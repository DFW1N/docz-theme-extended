# Header

Improved and redesigned header now allows adding of a logo image.

## Config

### Fixed header

Inside `doczrc.js` there is a `themeConfig.header` option:

```js
//doczrc.js
export default {
  themeConfig: {
    header: {
      fixed: true,
    },
  },
};
```

#### Options

| Name  | Type      | Default | Description                                       |
| ----- | --------- | ------- | ------------------------------------------------- |
| fixed | `boolean` | false   | Specify if the header should be fixed at the top. |

<hr />

### Logo

Inside `doczrc.js` there is a `themeConfig.logo` option:

<Hint type="warning">
  The image has to be placed inside the <InlineCode>public</InlineCode> folder
  or the provided options has to be an url address.
</Hint>

```js
//doczrc.js
export default {
  themeConfig: {
    logo: {
      src: '/public/logo.svg',
      width: 50,
    },
  },
};
```

Example with an url:

```js
//doczrc.js
export default {
  themeConfig: {
    logo: {
      src: 'https://www.my-website.com/image.jpg',
      width: 50,
    },
  },
};
```
