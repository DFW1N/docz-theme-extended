# Header

Improved and redesigned header now allows adding of a logo image.

## Config

Inside `doczrc.js` there is a `themeConfig.logo` option:

<Hint type="warning">
  The image has to be placed inside the <InlineCode>public</InlineCode> folder
  or the provided options has to be an url address.
</Hint>

```jsx
//doczrc.js
export default {
  themeConfig: {
    logo: {
      src: '/public/logo.svg',
      width: 50,
    },
  },
};
```

Example with an url:

```jsx
//doczrc.js
export default {
  themeConfig: {
    logo: {
      src: 'https://www.my-website.com/image.jpg',
      width: 50,
    },
  },
};
```
