# Footer

Added option to add footer to documents with different options.

- Navigation
- Updated date

## Config

### Navigation

Inside `doczrc.js` there is a `themeConfig.footer` option.

<Hint type="info">
  The navigation setting can be overriden inside the document itself with the{' '}
  <InlineCode>navigation</InlineCode> option.
</Hint>

```jsx
//doczrc.js
export default {
  themeConfig: {
    footer: {
      navigation: true,
    },
  },
};
```

```mdx
---
name: Footer
route: /ui/footer
menu: UI
navigation: false
order: 1
---

# My mdx file
```

### Order

To override the menu and navigation order you can now specify the order field inside your mdx file:

```mdx
---
name: Footer
route: /ui/footer
menu: UI
updated: 2020-05-15 12:00:00
order: 2
---

# My mdx file
```
