# Menu

Improved and redesigned menu.

- Right side menu with scroll detection
- Menu group label

## Config

### Menu

Inside `doczrc.js` there is a `themeConfig.menu` option.

```jsx
//doczrc.js
export default {
  themeConfig: {
    menu: {
      headings: {
        rightSide: true,
        scrollspy: true,
        depth: 3,
      },
    },
  },
};
```
